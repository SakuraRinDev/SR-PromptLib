Dockerとは？

Dockerは、アプリケーションをコンテナという軽量で移植可能な単位にパッケージ化するプラットフォームです。 「開発環境では動くのに本番では動かない」という問題を解決し、どこでも同じように動作する環境を提供します。

なぜDockerを使うのか？

1

環境の一貫性

開発、テスト、本番環境で全く同じコンテナを使用できるため、 環境差異によるバグを削減できます。

2

高速な起動

仮想マシンと比較して、コンテナは数秒で起動します。 OSを仮想化しないため、リソース効率も優れています。

3

スケーラビリティ

コンテナの複製や削除が簡単なため、 負荷に応じて柔軟にスケールアップ・ダウンできます。

インストール手順

ステップ1: システム要件の確認

Windows 10 Pro/Enterprise (64-bit)、macOS 10.14以降、 またはLinux (64-bit, カーネル3.10以降) が必要です。

ステップ2: Docker Desktopのダウンロード

公式サイトから各OS用のインストーラーをダウンロードし、 ウィザードに従ってインストールします。

ステップ3: 動作確認

# バージョン確認 docker --version # Hello Worldコンテナ実行 docker run hello-world

基本コマンド集

# イメージの検索と取得 docker search nginx docker pull nginx:latest
# コンテナの起動（-d: バックグラウンド, -p: ポートマッピング） docker run -d -p 8080:80 --name my-nginx nginx
# コンテナの管理 docker ps # 実行中のコンテナ一覧 docker ps -a # 全てのコンテナ一覧 docker stop my-nginx # コンテナ停止 docker start my-nginx # コンテナ開始 docker rm my-nginx # コンテナ削除
# イメージの管理 docker images # イメージ一覧 docker rmi nginx # イメージ削除

ベストプラクティス

複雑なアプリケーションも簡単に管理できます。
